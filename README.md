## Memulai

1. Clone repository:
   ```bash
   git clone https://github.com/yourusername/wbp-catalog.git
   cd wbp-catalog
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Jalankan development server:
   ```bash
   npm run dev
   ```

4. Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
